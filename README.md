# AINotes - UWP
Download: https://www.microsoft.com/de-de/p/ainotes/9ngx8jnlllcj

## Getting started
### Prerequesites
 - Visual Studio 2022

### Your first build
1. Clone the Repository
2. Open `AINotes/AINotes.sln` with Visual Studio
3. Set the Build Mode to "Debug" and the target to "Package"
4. Build & Run
